## 10-Line Summary

1. **Plan Before Coding**: Making AI plan before coding leads to faster shipping than jumping straight into implementation.

2. **Parallel Research Operations**: Five agents researching in parallel is much faster than human step-by-step planning, with humans providing taste, judgment, and product context.

3. **Specialist Agents**: Create domain-specific agents like "@event-tracking-expert" that distill everything about particular aspects of your codebase.

4. **Prevent Reinvention**: Search the codebase first to extend existing patterns instead of creating new ones from scratch.

5. **Source Code Reading**: Have agents read library source code directly when documentation lags behind updates, ensuring access to latest features.

6. **Three Automated Review Agents**: Automatically flag over-engineering, check for security vulnerabilities (like SQL injection), and enforce personal coding preferences.

9. **Decision-Making Framework**: Agents bring findings, humans make decisions, then collaborate to combine everything into one coherent plan.
